The Code for Hundo.

                        
                            //Get the values from the page
                            //Starts our controller function
                            function getValues() {
                                //Get values from the page
                                let startValue = document.getElementById("startValue").value;
                                let endValue = document.getElementById("endValue").value;
                            
                                //We need to validate our inputs
                                //Parse into Intergers
                                startValue = parseInt(startValue);
                                endValue = parseInt(endValue);
                            
                                if (Number.isInteger(startValue) && Number.isInteger(endValue)){
                                    //Call generateNumbers
                                   let numbers = generateNumbers(startValue, endValue);
                                    //Call displayNumbers
                                    displayNumbers(numbers);
                                } else {
                                    alert("You must enter Integers");
                                }
                            }
                            
                            //Generate numbers from the startValue to the endValue
                            //Logic function(s)
                            function generateNumbers(sValue, eValue) {
                                let numbers = [];
                            
                                //We want to get all numbers from start to end
                                for (let index = sValue; index <= eValue; index++) {
                            
                                    //This will run in a loop until index = eValue
                                    numbers.push(index);
                                }
                                return numbers;
                            }
                            
                            //Display numbers and mark even numbers in bold
                            //Display or view function(s)
                            function displayNumbers(numbers) {
                            
                                let templateRows = "";
                            
                                for (let index = 0; index < numbers.length; index++) {
                            
                                    let className = "even";
                            
                                    let number = numbers[index];
                            
                                    if (number % 2 == 0) {
                                        className = "even";
                                    } else {
                                        className = "odd";
                                    }
                                    
                                    templateRows += `<tr><td class="${className}">${number}</td></tr>`
                                }

                                document.getElementById("results").innerHTML = templateRows;
                            }

The Code is structured in three functions.

getValues( )

This is our controller function, and it is responsible for getting values the user has entered, from the webpage.

This function also allows us to parse our inputs into integers and then validate them.

Finally, this function calls our other two functions, seen lower in our code, to generate and display the numbers back to the user.
If no numbers have been entered, an alert will appear to the user, informing them that integers must be entered to use the app.

generateNumbers( )

Our second function is responsible for our logic.
It generates all the numbers from the starting point of zero, to the end point the user has determeined, with an upper limit of 100.

Finally these numbers are pushed to a variable and returned as an array for our first function, getValues(), to make use of.

displayNumbers( )

Our third function handles our display, once called by our getValues() function. This displays the values to the user in a table, with all even numbers bolded.

This function also makes use of template literals, to dynamically add HTML into the table, using JavaScript.
